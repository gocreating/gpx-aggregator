# 說明

使用者匯入本地gpx軌跡檔案。

# 觸發時機

使用者位於 `/` 頁面，且點選`工具列`中的`選擇資料夾`按鈕。

# 預期行為

- `工具列`中列出所有的匯入檔案資訊
  - `工具列`的軌跡有 "隱藏/顯示" 按鈕，可用來隱藏或顯示地圖上的軌跡
- 將使用者匯入的所有軌跡繪製於`地圖`上
- `工具列`與`地圖`的軌跡會雙向連動 hover 與 active 狀態
- `工具列`與`地圖`的軌跡轉變為active狀態時，在`地圖`上導航至該軌跡所在位置，然後縮放至適當大小
- hover `地圖`上的軌跡時，以 Tooltip 呈現該軌跡的花費時間、總距離、總爬升、總下降、高度落差
